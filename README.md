# MSch

MSch ("Matlab Secure CHannel") is an SSH/SCP/SFTP client library for Matlab.
It is built on top of the [JSch](http://www.jcraft.com/jsch/) Java Secure Channel library.

MSch is pronounced "mush".

## Requirements

Matlab R2019b or newer.
